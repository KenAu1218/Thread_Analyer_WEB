How to Use

1. Find a Post:

Go to Threads.net and find a conversation you want to analyze.

2. Copy URL:

Click the three-dot menu (...) on the post and select "Copy link".

(Alternatively, you can copy the URL from your browser's address bar.)

3. Analyze:

Paste it into the input box and click "Analyze".

4. Explore:

View the sentiment score of the main post and click on any reply to analyze that specific conversation thread recursively.

Note: Currently, we only support English text-based threads. Scores for other languages may not be accurate. The analysis is powered by the "cardiffnlp/twitter-roberta-base-sentiment-latest" model.
Note: The "Production/Debug" toggle allows developers to switch between the live server and a local testing server.
